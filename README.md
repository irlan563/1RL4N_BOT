# NfQ-BoT
Simple WhatsApp Bot

### FOR TERMUX USER
```bash
> pkg update && pkg upgrade
> pkg install git
> pkg install nodejs
> pkg install ffmpeg
> pkg install imagemagick
> git clone https://github.com/irlan563/1RL4N_BOT
> cd 1RL4N_BOT
> npm install
```
###### Run
```bash
> node . [<session name>] (session name is optional)
```

---------

### FOR WINDOWS/VPS/RDP USER
```bash
> git clone https://github.com/irlan563/1RL4N_BOT
> cd 1RL4N_BOT
> npm install
```
###### Run
```bash
> node index.js
```
